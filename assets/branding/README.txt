Place your app icon PNG here as "leveltrack_icon.png".

Requirements:
- Size: 1024×1024 PNG recommended (transparent background OK).
- Centered symbol, generous padding; avoid text inside the icon.
- Simple shapes, 2–3 colors for best scaling.

After adding the file, run:
1) flutter pub get
2) flutter pub run flutter_launcher_icons:main

This will generate Android adaptive icons, iOS icons, and Web icons.
